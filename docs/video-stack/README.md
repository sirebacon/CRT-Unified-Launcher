# PC to CRT Video Stack

## Overview

- Display: Sony Trinitron 27" CRT TV KV-27FV300 (configured for 240p/480i)
- Signal chain: PC -> StarTech HDMI2VGA -> OSSC (VoidScaler firmware) -> HDMI-to-YPbPr converter -> CRT
- PC output resolution: 1280 x 960
